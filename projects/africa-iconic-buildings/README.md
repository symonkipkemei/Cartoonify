# Africa's Iconic Buildings — Cartoonify Demo

10 buildings from BBC's *Africa's Iconic Architecture in 12 Buildings* (2021), cartoonified using the Cartoonify pipeline with the `gdo_cartoon` LoRA.

**Published:** [Cartoonify: Buildings as Political Objects](https://blog.iaac.net/cartoonify-buildings-as-political-objects/) — IAAC Blog, 2025

## Files

Each building has four files:

| Suffix | Description |
|---|---|
| `_original.png` | Source photograph (in `originals/`) |
| `_1024.png` | Source resized to 1024×1024 (pipeline input) |
| `_cartoon.png` | Cartoonified output |
| `_story.txt` | Two-sentence satirical caption (Gemini input) |
